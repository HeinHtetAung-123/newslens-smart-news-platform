# NewsLens: Smart News Aggregation and Bias Insight Platform

## Project Overview

NewsLens is a web-based smart news aggregation platform designed to help users access, understand, and compare news from multiple sources in one place. The platform allows users to browse news by category, search for specific stories, read quick summaries, save articles for later, and receive breaking news alerts.

A key feature of NewsLens is multi-source story comparison. This allows users to compare how different news sources report the same story and gain insight into differences in tone, focus, balance, and possible bias. The project aims to support a more organised, efficient, and balanced news-reading experience.

## Project Aim

The aim of this project is to develop a modern news website that helps users read news from multiple sources, understand articles quickly, and compare different perspectives on the same story.

## Main Problem

Many readers struggle to keep up with current news because information is spread across different websites and platforms. Articles can also be long, repetitive, or written from different perspectives. This makes it difficult for users to quickly understand a story and form a balanced view.

## Proposed Solution

NewsLens will collect and organise news articles from multiple sources. Users will be able to browse by category, search for topics, read short summaries, save articles, and view breaking news alerts. The platform will also provide comparison features to help users understand how different sources report the same event.

## Key Features

* Multi-source news feed
* News categories such as politics, technology, sports, business, entertainment, and local news
* Quick article summaries
* Breaking news alerts
* Search and filtering
* Save/bookmark articles
* Same-story source comparison
* Bias, tone, and balance insights
* User account management
* Admin management for categories, sources, and alerts

## Technology Stack

The planned technology stack may include:

* Frontend: HTML, CSS, JavaScript, Bootstrap
* Backend: Python Django
* Database: MySQL
* Version Control: Git and GitHub
* News Data: News API or RSS feeds
* Deployment: PythonAnywhere, Render, Railway, or similar platform
* Diagram Tools: Gliffy, GenMyModel, NinjaMock, or similar online tools

## Agile Development Approach

This project will be developed using an agile iterative approach. The work will be organised into three iterations under Milestone 1. Each iteration will include planned user stories, task tracking, implementation evidence, testing, reflection, and progress monitoring.
